◼ Elegiste aprender desarollo movil

Diego, al adentrarte en el desarrollo móvil después de haber aprendido Java, enfrentas un paisaje
tecnológico en constante cambio. Aunque Java fue una vez el principal lenguaje para el desarrollo
de aplicaciones Android, el panorama actual favorece a Kotlin, un lenguaje más moderno y con características avanzadas.
Sin embargo, tu experiencia en Java no es en vano. Java te ha proporcionado una base sólida en conceptos de programación que son transferibles y valiosos.
Además, entender Java te da una ventaja en el aprendizaje de Kotlin, dado que ambos lenguajes se utilizan en el ecosistema de Android y comparten similitudes.


Diego, tras aprender desarrollo de aplicaciones móviles con Java, te has equipado con una base sólida en uno de los lenguajes
históricamente fundamentales para Android. Aunque el entorno tecnológico evoluciona, tu conocimiento de Java sigue siendo valioso,
especialmente en el mantenimiento y la actualización de aplicaciones existentes. Ahora, mirando hacia el futuro, tienes varias opciones
para continuar tu crecimiento profesional

  • Aprender Kotlin
  • Dar soporte a aplicaciones móviles en Java
  • Regresar
